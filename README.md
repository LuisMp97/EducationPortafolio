# GIS Work Portfolio

- Overview

This portfolio showcases GIS and spatial programming work by Luis Maldonado, organized into professional and academic projects. It demonstrates hands-on experience with remote sensing, spatial ETL, database management, flood modelling, and custom geoprocessing tool development across a range of industry-standard platforms including Google Earth Engine, FME Workbench, ArcGIS, GRASS GIS, and Oracle Database.
Structure

- Structure

The portfolio is divided into two sections. The Work Portfolio contains two Google Earth Engine scripts developed for forest monitoring in Caquetá, Colombia, one performing a Random Forest-based forest/non-forest classification using Sentinel-2, Sentinel-1 SAR, and SRTM data, and another analyzing farm-level deforestation using the Hansen Global Forest Watch dataset. The Education Portfolio contains four projects developed during the COGS GIS program: a custom ArcPy geoprocessing tool for customer distribution analysis, two FME Workbench projects covering transit data ETL and Nova Scotia land cover spatial analysis, a Python-based flood mapping workflow using GRASS GIS with an accompanying poster for Richmond County, and an Oracle database project that loads and normalizes Statistics Canada crop data to Third Normal Form.
